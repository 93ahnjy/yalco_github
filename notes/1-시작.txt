1. git init
  - git 관리 내역을 저장하는 .git 폴더 생성

2. git status
  - 현재 폴더의 상황을 git의 상황으로 보여줌. 이 때  . git이 없다면 git status를 볼 수 없다.


3. .gitignore
  - git에 넣지 않을 파일, 폴더 포함. 그냥 .gitignore라는 이름의 파일을 만들고, 안에 파일, 폴더 목록을 넣으면 됨.
  - 보안 상 문제있거나 빌드 결과, 라이브러리 들을 여기에 넣을 수 있음
  - 형식은 https://git-scm.com/docs/gitignore 참조
